POS lock price or discount

This module add features to lock change price or discount in Point of Sale using password

Set configuration for Point of sale

Go to Point of Sale → Configuration → Point of Sale.
In Lock price or discount section, set Lock price or Lock discount

Check password in Point Of Sale

When the user click on Price or Discount, a popup will ask password

If the password is right, the user is able to change price or discount

If the password is wrong, a popup will appear
